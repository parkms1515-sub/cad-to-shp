# CAD to SHP Web Converter (v1.0)

이 프로젝트는 수치지형도 및 지적도를 기반으로 캐드 도면(DXF) 데이터를 공간정보(GIS) 데이터인 쉐이프파일(SHP)로 변환해 주는 Streamlit 기반의 웹 어플리케이션입니다.

## 🌟 주요 기능
웹 브라우저 상에서 DXF 파일을 업로드하고, 레이어를 선택하여 원하는 형태의 폴리곤 객체(SHP)로 자동 변환 및 지도 시각화를 제공합니다.

1. **DXF 데이터 업로드 및 분석**
   - DXF 파일(2007 버전 이하 권장) 업로드 지원
   - 도면층(Layer) 선택 및 추출된 객체의 화면 미리보기 제공
   - 객체 유형(Line, Point 등)에 따른 자동 분석

2. **국내 주요 좌표계(EPSG) 변환 지원**
   - 중부원점, 서부원점, 동부원점 등 지역별 측지계 가이드 제공
   - EPSG:5186, 5174, 5179, 5185 등 9개 이상의 국내 표준 좌표계 및 투영법(PRJ) 지원

3. **4가지 맞춤형 객체 변환 모드**
   - ⓐ **영역을 폴리곤으로 변환**: 폐합된 라인을 찾아 면(Polygon) 데이터로 변환합니다.
   - ⓑ **라인을 버퍼 적용(도로)**: 라인(선) 데이터에 지정한 거리만큼 버퍼를 주어 폴리곤으로 변환합니다.
   - ⓒ **포인트-라인-폴리곤 변환**: 다수의 측량 포인트를 순서대로 이어 점->선->면으로 단계별 변환합니다.
   - ⓓ **포인트를 폴리곤으로 변환(전주)**: 포인트 데이터를 기준으로 지정한 반경의 8각형 폴리곤을 생성합니다.

4. **실시간 지도 오버레이 (Web GIS)**
   - 변환된 데이터를 OpenStreetMap(일반 지도) 및 고해상도 위성 영상(Satellite) 위에 오버레이하여 정확성을 시각적으로 검증할 수 있습니다.
   - 화면 분할 모드를 통해 두 지도를 동시에 비교할 수 있습니다.

5. **SHP 파일 다운로드**
   - 변환된 결과를 공간정보 포맷인 `.shp`, `.shx`, `.dbf`, `.prj` 가 포함된 ZIP 파일 형태로 바로 다운로드할 수 있습니다.

## 🛠️ 설치 및 실행 방법

### 1. 요구 사항 (Prerequisites)
Python 3.x 환경이 필요하며, 아래의 패키지가 설치되어 있어야 합니다.

```bash
pip install -r requirements.txt
```

주요 의존성 라이브러리는 다음과 같습니다:
- `streamlit`: 웹 구현 및 UI
- `ezdxf`: 캐드 파일(DXF) 파싱 및 분석
- `pyshp` (shapefile): 공간 데이터(SHP) 생성
- `folium` / `streamlit-folium`: 지도 시각화
- `pyproj`: 좌표계 및 투영 변환

### 2. 실행 (Run)
터미널 혹은 명령 프롬프트에서 `app.py`가 있는 경로로 이동한 뒤, 아래 명령어를 실행합니다.

```bash
streamlit run app.py
```

명령어 실행 후 브라우저가 자동 실행되며 `http://localhost:8501`에서 어플리케이션을 이용할 수 있습니다.

## ⚠️ 작업 전 주의 사항
* 반드시 **DXF 2007 이하 버전**의 캐드 파일을 사용해 주세요.
* 도면에는 수치지형도 및 지적도의 실제 좌표계 정보가 반영되어 있어야 올바른 위치에 오버레이 됩니다.
* 빠르고 정확한 변환을 위해 하나의 도면층(Layer)에는 라인(Line) 혹은 포인트(Point) 등 **단일 객체 유형**만 포함되도록 모델링하는 것을 권장합니다.
